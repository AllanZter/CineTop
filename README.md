# 🎬 CineTop

Projeto desenvolvido como parte do curso de **Desenvolvimento JavaScript** do **SENAI**.

O CineTop é uma aplicação web inspirada em plataformas de streaming, que consome dados da API do TMDB para exibir filmes por categorias, lançamentos e permitir busca dinâmica.

> ⚠️ **Projeto em andamento**  
> Este projeto ainda está em desenvolvimento e pode receber melhorias, novas funcionalidades e ajustes visuais.

---

## 📚 Sobre o Projeto

Este projeto foi criado com o objetivo de aplicar na prática os conhecimentos adquiridos durante o curso, incluindo:

- Manipulação do DOM
- Consumo de API REST
- Organização de código JavaScript
- Estilização moderna com Tailwind CSS
- Estruturação de layout responsivo

A aplicação permite:

- 🎥 Exibir filmes por categoria (Drama, Ação, Comédia, Animação, etc.)
- 🔥 Mostrar lançamentos
- 🔎 Pesquisar filmes pelo nome
- 🎞️ Navegação horizontal estilo streaming
- 📱 Layout responsivo

---

## 🚀 Tecnologias Utilizadas

- **HTML5** – Estrutura da aplicação  
- **Tailwind CSS** – Estilização moderna e responsiva  
- **JavaScript (ES6+)** – Lógica da aplicação  
- **Fetch API** – Consumo de dados da API  
- **TMDB API** – Fonte de dados dos filmes  

---

## 🧠 Conceitos Aplicados

- Manipulação dinâmica de elementos com `createElement`
- Requisições HTTP com `fetch`
- Uso de funções reutilizáveis
- Eventos (`click`, `keypress`)
- Scroll horizontal programático
- Organização de múltiplas seções com IDs dinâmicos
